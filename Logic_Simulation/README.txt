Lab 1: Intro to Logic Simulation

Description:
In this lab we use Multimedia Logic to implement logic based on given truth tables.

Files:
Lab1.lgi - This file contains the Multimedia Logic program where the simulation for this lab is run

Instructions:
Click the green arrow on the top of the page in the Lab1.lgi file to run the program. While it is running,
click on each of the switches to see how each of the lights turn on/off in accordance with the given truth tables. 
Click the red box to stop running the simulation.
